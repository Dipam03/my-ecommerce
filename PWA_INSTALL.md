# PWA Installation Guide

## Install as Mobile App

Your Crodyto app is now a Progressive Web App (PWA) and can be installed on mobile devices just like a native app.

### On Android (Chrome/Edge)

1. Open the app in your mobile browser
2. Tap the **menu icon** (three dots) in the top right
3. Tap **"Install app"** or **"Add to home screen"**
4. Confirm the installation
5. The app will appear on your home screen

### On iPhone/iPad (Safari)

1. Open the app in Safari
2. Tap the **Share button** at the bottom
3. Scroll down and tap **"Add to Home Screen"**
4. Name the shortcut (or keep default "Crodyto")
5. Tap **"Add"**
6. The app will appear on your home screen

### Features

- **Offline Support**: Browse previously viewed products even without internet
- **Fast Loading**: App caches static files for faster performance
- **Home Screen Access**: Launch directly from home screen like a native app
- **App-like Experience**: Full-screen, no address bar, native navigation
- **Installable**: No need to install from app stores

### After Installation

- The app will update automatically in the background
- Works on Android, iPhone, iPad, and web browsers
- Takes up minimal storage compared to native apps
- Can be uninstalled like any other app

### Troubleshooting

**No install prompt?**
- Make sure you're on a mobile device or using mobile emulation
- Try adding it manually via Share (iPhone) or Menu (Android)

**App not loading offline?**
- The first visit needs to load online to cache files
- Subsequent visits will work offline (partially)
- Requires internet for Firebase/database operations

**Clear app cache?**
- On Android: Settings → Apps → Crodyto → Storage → Clear Cache
- On iPhone: Settings → Safari → Clear History and Website Data
